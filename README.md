# ZiZiBOOKS

シンプルかつ高機能な、エンジニア・クリエイター向けWebランチャー/ブックマーク管理ツール。

## 概要

ZiZiBOOKSは、日常的に使用するURLやローカルパスを整理し、素早くアクセスするためのWPFアプリケーションです。
「セミモード（リスト形式）」と「マジモード（タイル形式）」の2つのUIを切り替えて使用でき、ドラッグ＆ドロップによる直感的な並び替えに対応しています。

## 主な機能

- **デュアルUIモード**: 
  - **Semi Mode**: アイコンとテキストが並ぶ標準的なリスト形式。
  - **Maji Mode**: 作業に集中するための、シンプルで視認性の高いタイル形式。
- **直感的なD&D**: メイン画面および設定画面の両方で、項目の並び替えが可能。
- **スマートクリック判定**: 微細なマウスの動きでドラッグと誤判定されるのを防ぐ、独自のクリック救済ロジックを搭載。
- **自動保存**: ウィンドウ位置、フォントサイズ、最前面表示設定などを終了時に自動保持。
- **ポータビリティ**: 設定はJSON/DICTファイルとして保存され、インポート/エクスポートにも対応。

## 技術スタック

- **Framework**: .NET 9 / WPF
- **Language**: C# 13
- **IDE**: Visual Studio 2022/2026
- **Dependencies**: 
  - System.Text.Json (設定のシリアライズ)
  - Win32 API (アイコンの抽出)
