# ZiZiBOOKS

シンプルかつ高機能な、エンジニア・クリエイター向けWebランチャー/ブックマーク管理ツール。

## 概要

**ZiZiBOOKS** は、シンプルさと機能性を両立させた Windows 用のデスクトップランチャーです。
「セミモード（リスト形式）」と「マジモード（タイル形式）」の2つのUIを切り替えて使用でき、ドラッグ＆ドロップによる直感的な並び替えに対応しています。
登録数が増えてもデスクトップを圧迫せず、高解像度（4K）環境ではその広さを最大限に活かすダイナミックなレイアウトエンジンを搭載しています。

## ✨ 特徴

- **デュアルUIモード**: 
  - **Semi Mode**: アイコンとテキストが並ぶ標準的なリスト形式。
  - **Maji Mode**: 作業に集中するための、シンプルで視認性の高いタイル形式。
- **解像度追従型レイアウト**: 
  - FHD以下の環境では、画面を突き抜けないよう作業領域の65%を上限に自動スクロール。
  - 4K環境では、登録数に合わせて制限を解除し、一覧性を最大化。
- **OLED焼き付き防止機能**: 放置時間に合わせてウィンドウを自動で減光（フェードアウト）。
- **ドラッグ＆ドロップ登録**: URLやローカルファイルを枠内に落とすだけで即座に登録。
- **直感的な並び替え**: メイン画面上でボタンをドラッグするだけで順序を整理。
- **マルチモード**: リスト形式の「Semi Mode」と、タイル形式の「Maji Mode」を切り替え可能。
- **スマートクリック判定**: 微細なマウスの動きでドラッグと誤判定されるのを防ぐ、独自のクリック救済ロジックを搭載。
- **自動保存**: ウィンドウ位置、フォントサイズ、最前面表示設定などを終了時に自動保持。
- **ポータビリティ**: 設定はJSON/DICTファイルとして保存され、インポート/エクスポートにも対応。


## 🚀 インストール / 実行方法

1. [Releases](https://github.com/your-repo/ZiZiBOOKS/releases) から最新のバイナリをダウンロードします。
2. 解凍したフォルダ内の `ZiZiBOOKS.exe` を実行してください。
   - ※本アプリは `.NET 9.0` を使用しています。未インストールの場合は[こちら](https://dotnet.microsoft.com/download/dotnet/9.0)からランタイムをインストールしてください。

## 🛠 使い方

- **設定を開く**: タイトル右側の歯車アイコンをクリック。
- **項目の追加**: 設定パネルの入力欄に情報を入力し「追加/更新」を押すか、エクスプローラーからファイルをドロップ。
- **項目の編集**: 設定パネル内の「現在のブックマーク」リストを編集ボタンで選択。
- **項目の削除**: 設定パネル内のリストから「消」ボタンをクリック。
